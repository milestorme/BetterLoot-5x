### BetterLoot 5x 
 
This is a 5x default setup with optimised loot removing most junk items.

To use this plugin, you need to download and install the latest version of the free BetterLoot plugin:
👉 https://umod.org/plugins/better-loot

### INSTALLATION INSTRUCTIONS

1. Place the following 2 files located inside the data folder into your oxide/data/BetterLoot folder replacing default ones.

- LootGroups.json
- LootTables.json

2. Place the following file located inside the config folder into your oxide/config/BetterLoot folder replacing the default one.

- BetterLoot.json

3. Reload the plugin:

o.reload BetterLoot

Note: You can modify this further at 👉 https://looty.cc/betterloot
